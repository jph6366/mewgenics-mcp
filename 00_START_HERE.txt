THE 6 MCP TOOLS YOU'LL BUILD
================================================================================

1. analyze_save_file
   → Loads .sav file, returns cat stats, breeding opportunities
   
2. generate_breeding_pair
   → Suggests optimal pair for goal (max DPS, pure bloodline, etc)
   
3. generate_combat_team
   → Recommends 3-cat teams with synergy scores and strategy
   
4. rank_cats_for_breeding
   → Scores all cats by breeding potential + compatible partners
   
5. rank_teams_for_battle
   → Ranks all possible team combos by synergy + win probability
   
6. simulate_breeding_outcome
   → Monte Carlo simulation (1000 trials) of offspring traits

Each tool is called by Claude (or other LLMs) and returns JSON results.
The server speaks with an erratic, cosmic horror persona throughout.

================================================================================
THE NOVEL FEATURE: CONTEXT ROT TRACKING
================================================================================

Problem: AI assistants degrade over long conversations
- Data becomes stale
- Recommendations contradict earlier advice
- Personality becomes generic
- Memory bloats (expensive tokens)

Solution: Measure context health across 4 dimensions:

1. DATA FRESHNESS (25% weight)
   How many turns since save file was reloaded?
   Turn 0: 100%, Turn 5: 80%, Turn 10: 60%, Turn 20: 40%

2. SEMANTIC COHERENCE (35% weight)
   Do breeding recommendations stay consistent?
   High overlap: 85%, Low overlap: 40%

3. PERSONA CONSISTENCY (25% weight)
   Is erratic tone maintained or devolving to bland LLM?
   Erratic: 100%, Bland: 0%

4. MEMORY SATURATION (15% weight)
   Is context window bloated?
   1000 tokens: 100%, 5000 tokens: 10%

OVERALL HEALTH = Weighted average (0-100%)

When health < 75%: ⚠️ "The void creeping in"
When health < 50%: 🚨 "THE WALLS ARE REMEMBERING WRONG THINGS"

User knows exactly when to reset. No guessing.

Test Results (5/6 passing):
- Health degrades from 97% (Turn 1) → 34% (Turn 20)
- Warnings trigger at appropriate thresholds
- Persona drift detected within 1-2 turns
- System recommends reset at exactly the right time
